# Home Assistant Add-on: Meshmonitor

## About

This repository contains a Home Assistant add-on for [MeshMonitor](https://github.com/yeraze/meshmonitor), allowing you to monitor and manage your Meshtastic mesh network directly from Home Assistant.

**Note**: This add-on was built for personal use and is shared as-is. You're welcome to use it, but I don't make any guarantees about how quickly I'll respond to issues or implement new features. Pull requests are welcome if you'd like to contribute improvements!

## Installation

Adding this add-ons repository to your Home Assistant instance is pretty straightforward. In the Home Assistant add-on store, a possibility to add a repository is provided.

Use the following URL to add this repository:

```txt
https://github.com/bhardie/ha-meshmonitor
```

[![Add Repository][repository-badge]][repository-url]

## Available Add-ons

This repository contains the following add-on:

### [Meshmonitor](./meshmonitor)

![Version](https://img.shields.io/badge/version-0.0.1-blue.svg)

Monitor and manage your Meshtastic mesh network from Home Assistant.
